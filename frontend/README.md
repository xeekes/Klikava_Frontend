# KlikAVA Frontend

React storefront for the KlikAVA online marketplace.

## Setup

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

Set `VITE_API_BASE_URL` in `.env` to your backend URL.

## Scripts

- `npm run dev` — local development
- `npm run build` — production build
- `npm run test` — unit tests
